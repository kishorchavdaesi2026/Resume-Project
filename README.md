# 🚀 Resume IQ

Resume IQ is a modern AI-powered Resume Builder developed using Next.js and TypeScript. It enables users to create professional resumes with an intuitive interface and export them in multiple formats.

---

## ✨ Features

- Modern Responsive UI
- Resume Creation & Editing
- Export Resume to PDF
- Export Resume to DOCX
- State Management
- Fast Performance
- Built with Next.js 15

---

## 🛠 Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS
- Node.js
- npm

# 🚀 Getting Started

## 1. Clone the Repository

```bash
git clone https://github.com/kishorchavdaesi2026/Resume-Project.git
```

---

## 2. Navigate to the Project Folder

```bash
cd Resume-Project
```

---

## 3. Go to the Next.js Project

```bash
cd resume-iq-next
```

---

## 4. Install Dependencies

```bash
npm install
```

---

## 5. Run the Development Server

```bash
npm run dev
```

---

## 6. Open in Browser

Visit

```
http://localhost:3000
```

Your Resume IQ application should now be running locally.

---

# 📦 Production Build

Create a production build:

```bash
npm run build
```

Start the production server:

```bash
npm start
```
